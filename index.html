bake

builds and concatenates static websites from a list of templates

Download this project as a .zip file Download this project as a tar.gz file

bake is ruby-based

'bake' will assemble a web site give the specified build file (usually bakefile.json); concatenating the header/footer for each page set.

usage

Options:
  -e, --version     outputs version and exit
  -f, --file        bakefile (if specifying build file other than bakefile.json)
  -h, --help        displays this help and exit
  -p, --push        pushes website out to destination (-d required if bakefile.json not present)
  -d, --dest        destination [protocol:server:uid:pwd]
  -v, --verbose     outputs debugging information
  -w, --helloworld  outputs example site
  -l, --log <file>  captures all output to file

sample bakefile.json

{
    /* bakefile.json - example build file */

    "exclude_files" : "",

    "pre_execute" : "",

    "post_execute" :  "",

    "output_dir" : "website",

    "static_dir" : "static",

    "header": "header.html",

    "footer": "footer.html",

    "pages" : ["index.html", "home.html"],

    "flags" : {

        "publish" :
        {
            "publish" : "true",
            "archive" : "true",
            "publish_server" : "ftp.server.com",
            "publish_type" : "ftp",
            "uid" : "uid",
            "pwd" : "pwd"
        }
    }
}
